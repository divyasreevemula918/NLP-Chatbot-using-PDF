# 📄 PDF Question Answering Chatbot

An intelligent **PDF Question Answering Chatbot** built using **Streamlit**, **LangChain**, **Google Gemini API**, and **FAISS**.  
This application allows users to upload PDF files, ask questions from the uploaded content, and get accurate answers in a chatbot-style interface.

## 📌 Project Overview

This project is designed to make interaction with PDF documents easier and smarter.  
Instead of reading the whole document manually, users can simply:

- Upload one or more PDF files 📂
- Ask questions based on the PDF content ❓
- Receive relevant answers instantly 💡

It works like a mini **ChatGPT for PDFs**.

---

## ✨ Features

- 📤 Upload PDF documents
- 📚 Extract text from uploaded PDFs
- ✂️ Split text into chunks
- 🧠 Convert text into embeddings
- 🔍 Store embeddings using FAISS vector database
- 💬 Ask questions from the PDF
- 🤖 Get context-based answers using Gemini API
- 🖥️ Simple and interactive Streamlit UI

---

## 🛠️ Tech Stack

- **Python**
